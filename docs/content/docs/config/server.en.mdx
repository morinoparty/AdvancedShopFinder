## File location

The server configuration for Advanced Shop Finder is stored at `plugins/AdvancedShopFinder/config.json`. This file is generated automatically the first time the plugin starts.

## Configuration options

`config.json` contains the following options.

| Option | Type | Default | Description |
| --- | --- | --- | --- |
| `placeData` | List of places | Morimoto (1 entry) | Reference points used to compute the nearest town (near-town). |
| `fuzzySearchLimit` | Integer | `50` | The maximum number of results returned by fuzzy search. Fuzzy search can match many shops, so this value caps the result count. |
| `format` | String (MiniMessage) | See below | The display format for a single search result. |

### placeData

`placeData` is a list of place entries. Each entry has **only** the following three fields.

| Field | Type | Description |
| --- | --- | --- |
| `x` | Integer | The X coordinate of the place. |
| `z` | Integer | The Z coordinate of the place. |
| `placeName` | String | The name of the place. Supports MiniMessage. |

The places registered here are used to calculate the nearest town (near-town) for each shop and the distance to it.

:::note
Place entries do **not** have a `world` field. A place is defined solely by `x`, `z`, and `placeName`.
:::

## Default config.json

```json title="config.json"
{
  "placeData": [
    {
      "x": -532,
      "z": -85,
      "placeName": "もりもと"
    }
  ],
  "fuzzySearchLimit": 50,
  "format": "<shop-type>: オーナー:<green><player-name></green> 値段: <green><price>/<shop-stacking-amount></green>個 在庫: <green><count></green> \n座標: <yellow><world></yellow> x:<blue><x></blue> y:<blue><y></blue> z:<blue><z></blue> 距離: <green><distance></green>ブロック 最寄り: <near-town>から<green><near-town-distance></green>ブロック"
}
```

## About format

`format` is the display format for a single search result and supports the [MiniMessage](https://docs.advntr.dev/minimessage/format) format. The following placeholders are substituted in.

| Placeholder | Description |
| --- | --- |
| `<shop-type>` | Shop type: buying (red) / selling (green). |
| `<player-name>` | The shop owner's name. |
| `<price>` | The sale price. |
| `<shop-stacking-amount>` | The selling unit (e.g. 32 items). |
| `<count>` | Stock count (counted as 64 items per stack; 5 stacks is expressed as 5 × 64). |
| `<distance>` | Straight-line distance. |
| `<near-town>` | The nearest town. |
| `<near-town-distance>` | Straight-line distance from the center of the nearest town. |
| `<world>` | The shop's world name. |
| `<x>` | The shop's X coordinate. |
| `<y>` | The shop's Y coordinate. |
| `<z>` | The shop's Z coordinate. |

## Applying changes

:::warning
After editing `config.json`, the changes will not take effect until you run the `sf reload` command or restart the server. Running `sf reload` requires the `advancedshopfinder.admin` permission.
:::
