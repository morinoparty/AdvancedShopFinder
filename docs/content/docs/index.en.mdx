## What is Advanced Shop Finder?

Advanced Shop Finder is a shop search plugin for the Minecraft server plugin [QuickShop-Hikari](https://github.com/Ghost-chu/QuickShop-Hikari). With item search, enchanted book search, and fuzzy search, you can quickly find shops on your server.
