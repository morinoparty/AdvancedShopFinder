## 検索プロファイルとは

検索プロファイルは、検索の挙動（並び順・取得件数・在庫表示など）をまとめて保存した設定のセットです。プロファイルには名前を付けて複数作成でき、用途に応じて使い分けられます。

- 各プレイヤーは複数のプロファイルを持てます。初期状態では `default` プロファイルが用意されています。
- 現在「使用中」のプロファイルが、通常の検索で自動的に適用されます。
- 個別の検索では `-p <プロファイル名>` フラグで一時的に別のプロファイルを指定できます（使用中のプロファイルは変更されません）。

プロファイルの実体はプレイヤーごとに次の場所へ JSON として保存されます。各フィールドの詳細は[検索オプション](/ja/docs/config/find-options)を参照してください。

```text
plugins/AdvancedShopFinder/playerdata/<player-uuid>/config.json
```

## 設定 GUI

`/sf setting` を実行すると、現在使用中のプロファイルの設定 GUI（チェスト型インベントリ）が開きます。各項目はアイテムとして表示され、クリックして操作します。

| 項目 | 操作 |
| --- | --- |
| 買取ショップの並び順（buySortType） | 左クリックで次の並び順 / 右クリックで前の並び順 |
| 販売ショップの並び順（sellSortType） | 左クリックで次の並び順 / 右クリックで前の並び順 |
| 買取ショップの取得件数（buyFindLimit） | 左クリックで +1 / 右クリックで −1 / Shift クリックで無制限 |
| 販売ショップの取得件数（sellFindLimit） | 左クリックで +1 / 右クリックで −1 / Shift クリックで無制限 |
| 在庫切れショップを表示（showNoStockShop） | クリックで ON / OFF を切り替え |
| オーナーの所持金不足ショップを表示（showOwnerNotHasEnoughMoneyShop） | クリックで ON / OFF を切り替え |
| このプロファイルを使用中にする | クリックでこのプロファイルを使用中に設定 |
| 保存 | 変更を保存して閉じる |
| 閉じる | 保存せず閉じる |

:::note
取得件数の「無制限」は内部値 `-1` に対応します。並び順の値は SortType で表され、指定できる値の一覧は[検索オプション](/ja/docs/config/find-options)にまとめられています。
:::

:::warning
「閉じる」で閉じた場合、変更は保存されません。設定を反映するには必ず「保存」をクリックしてください。
:::

## `sf setting` コマンド

プロファイルの管理はすべてプレイヤー本人が実行できます（権限は不要です）。

| コマンド | 説明 |
| --- | --- |
| `sf setting` | 現在使用中のプロファイルの設定 GUI を開きます。 |
| `sf setting <profile>` | 指定したプロファイルの設定 GUI を開きます。存在しない場合は新規に作成します。 |
| `sf setting list` | すべてのプロファイルを一覧表示します。使用中のプロファイルには「使用中」と表示されます。 |
| `sf setting use <profile>` | 使用中のプロファイルを切り替えます。 |
| `sf setting delete <profile>` | プロファイルを削除します。`default` は削除できません。使用中のプロファイルを削除した場合は `default` に戻ります。 |

## `-p` フラグで検索ごとに切り替える

検索コマンド `sf search` / `sf search-book` / `sf fuzzy-search` は、`--profile <名前>` または短縮形 `-p <名前>` フラグを受け付けます。指定したプロファイルの設定でその検索のみを実行し、使用中のプロファイルは変更されません。

```text
sf search diamond -p mining
sf fuzzy-search dia --profile mining
```

:::warning
存在しないプロファイル名を指定すると、`プロファイル '<名前>' が見つかりません` というエラーが表示され、検索は実行されません。
:::
