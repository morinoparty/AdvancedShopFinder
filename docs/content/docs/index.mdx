## Advanced Shop Finder とは？

Advanced Shop Finder は、Minecraft サーバープラグイン [QuickShop-Hikari](https://github.com/Ghost-chu/QuickShop-Hikari) に対応したショップ検索プラグインです。アイテム検索・エンチャント本検索・あいまい検索を使って、サーバー上のショップを素早く見つけられます。
